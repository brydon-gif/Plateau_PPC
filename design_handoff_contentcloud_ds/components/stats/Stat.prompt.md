**Stat** — outcome figures. Big red Jura value, muted label. Note the site uses qualitative values ("Cited", "Fewer", "Faster", "5+") as often as numbers — keep values short.

```jsx
<Stat value="5+" label="Languages supported" />
<Stat value="Cited" label="Answers grounded in approved sources" />
```

Lay 3–4 in a row separated by whitespace or thin dividers. Use `align="left"` inside cards.
