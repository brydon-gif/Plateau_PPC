**Input** — text field / textarea for contact & sign-up forms.

```jsx
<Input label="Work email" placeholder="you@org.eu" type="email" />
<Input label="How can we help?" multiline rows={5} />
<Input label="Seats" invalid hint="Required" />
```

Focus draws the ink border + a soft red ring. Pair labels in Jura with body-font inputs. Corners are 6px (tighter than buttons).
